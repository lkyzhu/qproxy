# Overview

**qproxy** 是一个基于QUIC协议的加密SOCKS5代理，旨在提供安全可靠的网络通信。它利用QUIC（Quick UDP Internet Connections）作为其传输层，以确保高效和低延迟的数据传输。

**qproxy** 旨在防止数据在传输过程中被窃取或被链接监控，确保用户数据的私密性和安全性。
